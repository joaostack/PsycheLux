# PsycheLux
Windows Terminal theme based on color psychology

```json
{
    "background": "#0C0A12",
    "black": "#161320",
    "blue": "#4B6FDB",
    "brightBlack": "#2A2540",
    "brightBlue": "#6F92FF",
    "brightCyan": "#8FE9F5",
    "brightGreen": "#7FD2C7",
    "brightPurple": "#B28CFF",
    "brightRed": "#E06F87",
    "brightWhite": "#FFFFFF",
    "brightYellow": "#F2D48A",
    "cursorColor": "#D6B56D",
    "cyan": "#5FC7D8",
    "foreground": "#E8E4F2",
    "green": "#5FA89E",
    "name": "Psyche Lux",
    "purple": "#8F5EC7",
    "red": "#B74C63",
    "selectionBackground": "#26203A",
    "white": "#D8D3E6",
    "yellow": "#D6B56D"
}
```
